# Guía para Crear Screenshots de Zentavo

## 📸 Requisitos de Google Play Store

### Tamaños Requeridos
- **Mínimo**: 320 px en cualquier dimensión
- **Máximo**: 3840 px en cualquier dimensión
- **Relación de aspecto**: Entre 16:9 y 9:16
- **Formato**: PNG o JPG (PNG recomendado)
- **Cantidad**: Mínimo 2, máximo 8 screenshots

### Tamaños Ideales
Para smartphones modernos:
- **1080 x 1920 px** (Full HD, 9:16) - Recomendado
- **1080 x 2400 px** (20:9) - Para pantallas alargadas
- **1242 x 2688 px** (iPhone XS Max) - Compatible

## 🎨 Estrategia de Screenshots

### Screenshots Obligatorios (Orden Sugerido)

#### 1. **Pantalla Principal - Dashboard** 
**Qué mostrar:**
- Vista general con saldo actual
- Gráfico circular de gastos por categoría
- Tarjetas de ingresos/gastos/ahorros
- Fecha actual

**Mensaje sugerido (overlay):**
```
"Controla tus finanzas en un vistazo"
"Visualiza tu dinero de forma clara"
```

#### 2. **Gestión de Gastos**
**Qué mostrar:**
- Lista de transacciones recientes
- Diferentes categorías con íconos coloridos
- Filtros por mes visible
- Botón de agregar transacción

**Mensaje sugerido:**
```
"Registra gastos en segundos"
"Categoriza automáticamente tus compras"
```

#### 3. **Gráficos y Estadísticas**
**Qué mostrar:**
- Gráfico de barras comparativo por mes
- Gráfico de líneas de tendencias
- Estadísticas de categorías más gastadas

**Mensaje sugerido:**
```
"Analiza tus patrones de gasto"
"Toma decisiones informadas"
```

#### 4. **Ahorros y Metas**
**Qué mostrar:**
- Tarjetas de objetivos de ahorro
- Barra de progreso al 60-70% (visual atractivo)
- Diferentes monedas (USD, EUR)
- Botón de agregar nuevo ahorro

**Mensaje sugerido:**
```
"Alcanza tus metas financieras"
"Ahorra en múltiples monedas"
```

#### 5. **Presupuestos**
**Qué mostrar:**
- Presupuestos por categoría con barras de progreso
- Algunas categorías en verde (bajo presupuesto)
- Una categoría en amarillo/rojo (alerta de límite)
- Porcentajes visibles

**Mensaje sugerido:**
```
"Establece límites inteligentes"
"Evita gastos excesivos"
```

#### 6. **Recordatorios**
**Qué mostrar:**
- Lista de pagos programados
- Fechas próximas destacadas
- Íconos de categorías
- Estado de activado

**Mensaje sugerido:**
```
"Nunca olvides un pago"
"Recordatorios automáticos"
```

#### 7. **Exportar Reportes** (Opcional)
**Qué mostrar:**
- Diálogo de exportación
- Opciones de formato (Excel, PDF)
- Calendario de rango de fechas
- Vista previa de reporte

**Mensaje sugerido:**
```
"Exporta reportes profesionales"
"Formato Excel y PDF"
```

#### 8. **Multi-idioma y Personalización** (Opcional)
**Qué mostrar:**
- Menú de configuración
- Lista de idiomas disponibles
- Opciones de moneda
- Tema claro/oscuro

**Mensaje sugerido:**
```
"Disponible en 6 idiomas"
"Personaliza a tu gusto"
```

## 🛠️ Cómo Crear los Screenshots

### Opción 1: Desde la App Real (Recomendado)

**Preparación:**

1. **Limpia datos de prueba**
   ```
   - Elimina transacciones de test
   - Deja datos realistas pero no personales
   ```

2. **Llena con datos de ejemplo atractivos**
   ```
   Ingresos:
   - Salario: $3,500
   - Freelance: $800
   
   Gastos:
   - Supermercado: $450
   - Transporte: $120
   - Restaurantes: $180
   - Entretenimiento: $90
   
   Ahorros:
   - Vacaciones: $1,200 de $2,000 (60%)
   - Emergencias: $3,000 de $5,000 (60%)
   ```

3. **Configura la app**
   ```
   - Activa modo oscuro O claro (consistente en todas)
   - Selecciona idioma español
   - Configura moneda a tu región
   ```

**Captura:**

1. **Abre la app en tu dispositivo**
2. **Navega a cada pantalla**
3. **Toma screenshot**:
   - **Android**: Volumen abajo + Power
   - **iOS**: Power + Volumen arriba

4. **Transfiere a PC**:
   ```
   - Conecta por USB
   - O usa Google Photos / iCloud
   ```

### Opción 2: Usar Emulador Android Studio

**Ventajas:**
- Resolución perfecta
- Sin notificaciones molestas
- Hora/batería ideales

**Pasos:**

1. **Abre Android Studio**
2. **Inicia emulador**: Pixel 6 Pro (1080x2400)
3. **Instala APK**: Arrastra archivo a emulador
4. **Configura emulador**:
   ```
   - Cambia hora a 10:30 AM
   - Batería al 100%
   - Sin notificaciones
   - Modo avión (sin señal molesta)
   ```
5. **Captura**: Botón de cámara en barra lateral del emulador

### Opción 3: Device Art Generator (Mockups)

**Para hacer screenshots más profesionales:**

**Herramientas:**
- **Device Frame Generator**: https://developer.android.com/distribute/marketing-tools/device-art-generator
- **Mockuphone**: https://mockuphone.com
- **Canva**: Plantillas de mockup de apps

**Proceso:**
1. Sube tus screenshots simples
2. Coloca en marco de dispositivo (Pixel, Samsung)
3. Agrega fondo degradado
4. Añade texto overlay
5. Descarga resultado

## 🎨 Diseño y Estilo

### Overlays de Texto (Opcional pero Recomendado)

**Herramientas:**
- **Canva**: Plantillas gratuitas de Play Store
- **Figma**: Diseño profesional
- **Adobe Express**: Rápido y fácil

**Elementos a añadir:**
```
- Título corto arriba (max 5 palabras)
- Fondo semi-transparente detrás del texto
- Colores: Azul #2196F3 (color primario de Zentavo)
- Fuente: Sans-serif moderna (Roboto, Inter, Poppins)
- Tamaño texto: Grande y legible
```

**Ejemplo de diseño:**
```
┌─────────────────────────┐
│                         │
│  [Screenshot de App]    │
│                         │
│  ┌──────────────────┐   │
│  │ Texto Overlay    │   │
│  │ "Controla tu $"  │   │
│  └──────────────────┘   │
│                         │
└─────────────────────────┘
```

### Consistencia Visual

**Mantén uniforme:**
- ✅ Mismo marco de dispositivo en todas
- ✅ Mismo estilo de texto (si usas overlays)
- ✅ Mismo tema de app (claro o oscuro)
- ✅ Mismo idioma
- ✅ Misma moneda mostrada

**Evita:**
- ❌ Mezclar tema claro y oscuro
- ❌ Notificaciones personales visibles
- ❌ Datos reales/sensibles
- ❌ Hora/batería inconsistente
- ❌ Cambiar idioma entre capturas

## 📋 Checklist Final

Antes de subir a Play Store, verifica:

```
☐ Mínimo 2 screenshots (recomendado 4-6)
☐ Resolución correcta (1080x1920 o similar)
☐ Formato PNG (mejor calidad que JPG)
☐ Tamaño < 8MB cada uno
☐ Sin información personal visible
☐ Sin notificaciones extrañas
☐ Hora realista (10:00-14:00)
☐ Batería alta (80-100%)
☐ Señal completa o modo avión
☐ Datos de ejemplo atractivos
☐ Colores vibrantes y claros
☐ Funcionalidad clave mostrada
☐ Texto legible (si usas overlays)
☐ Mismo estilo en todas
☐ Orden lógico (dashboard primero)
```

## 🎯 Capturas Prioritarias para el Mínimo

**Si solo vas a hacer 2 screenshots** (mínimo de Play Store):

1. **Dashboard principal** - Muestra la visión general
2. **Gráfico de gastos** - Muestra valor de análisis

**Si vas a hacer 4** (recomendado):

1. Dashboard principal
2. Gráfico de gastos por categoría
3. Lista de transacciones con categorías
4. Pantalla de ahorros/metas

## 💡 Tips Profesionales

### Para Destacar en Play Store:

1. **Primera impresión**: El primer screenshot es el más importante
2. **Historia visual**: Cuenta una historia (problema → solución)
3. **Texto mínimo**: Si usas overlays, max 5 palabras por imagen
4. **Alta calidad**: Captura en dispositivo de alta resolución
5. **Localización**: Considera screenshots en inglés también

### Screenshots en Otros Idiomas:

**Para mercado internacional** (opcional):
- Captura mismas pantallas en inglés
- Sube como "screenshots en inglés" en Play Console
- Play Store mostrará versión correcta según idioma de usuario

## 📤 Formato de Entrega

**Nombra tus archivos:**
```
screenshot_01_dashboard.png
screenshot_02_grafico_gastos.png
screenshot_03_transacciones.png
screenshot_04_ahorros.png
screenshot_05_presupuestos.png
screenshot_06_recordatorios.png
```

**Organiza en carpeta:**
```
play_store_assets/
├── screenshots/
│   ├── es/
│   │   ├── 01_dashboard.png
│   │   ├── 02_grafico.png
│   │   └── ...
│   └── en/ (opcional)
│       ├── 01_dashboard.png
│       └── ...
├── icon_512x512.png
└── feature_graphic_1024x500.png
```
