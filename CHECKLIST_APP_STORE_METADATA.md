# Checklist para Metadata de App Store (Zentavo)

## 1. Enlaces legales en metadata
- [ ] Agregar URL de Política de Privacidad:
      https://zentavo.com/privacidad
- [ ] Agregar URL de Términos de Uso (EULA):
      https://zentavo.com/terminos

## 2. Pantalla premium
- [x] Enlaces visibles y funcionales a Política de Privacidad y Términos de Uso.
- [x] Mensaje: "Al suscribirte aceptas nuestros términos y políticas."

## 3. Información de suscripción
- [ ] Descripción clara de los beneficios premium en la metadata.
- [ ] Indicar que la suscripción se renueva automáticamente (si aplica).
- [ ] Explicar cómo cancelar la suscripción (desde App Store > Cuenta > Suscripciones).

## 4. Cumplimiento de políticas
- [x] No existen cupones, códigos de descuento ni bypass premium.
- [x] Premium solo se activa con compra real o restauración legítima.
- [x] No hay helpers ni flags de testing en producción.

## 5. Otros
- [ ] Revisar que los enlaces legales estén también en la web de soporte si Apple lo solicita.
- [ ] Revisar que la información de contacto esté actualizada.

---

**Este checklist debe completarse antes de enviar la app a revisión.**
