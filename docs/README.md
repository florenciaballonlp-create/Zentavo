# Zentavo - Privacy Policy

This directory contains the privacy policy webpage for Zentavo app.

## GitHub Pages

The privacy policy is published at: https://florenciaballonlp-create.github.io/Zentavo/

## How to Update

1. Edit `index.html` in this directory
2. Commit and push to GitHub
3. Changes will be live within minutes

## Local Preview

Open `index.html` in a web browser to preview changes before publishing.

## Required for Play Store

Google Play Store requires a publicly accessible privacy policy URL. This page fulfills that requirement.
