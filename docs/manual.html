Manual de Uso

Guia rapida para comenzar a usar Zentavo.

Este manual resume los pasos principales para organizar tus finanzas, registrar movimientos y usar funciones compartidas.

1. Configuracion inicial

  • Abre Zentavo y revisa la pantalla principal.
  • Define tu moneda principal desde configuracion.
  • Activa notificaciones si quieres recordatorios.

2. Registrar ingresos y gastos

  • Usa los botones Income y Expenses para agregar movimientos.
  • Completa monto, categoria y fecha.
  • Guarda y revisa el impacto en el balance total.

3. Usar ahorros

  • Crea metas de ahorro desde la seccion Savings.
  • Asigna montos periodicos y controla el progreso.

4. Eventos compartidos

  • En Shared Events crea un evento y comparte el codigo o QR.
  • Invita participantes para llevar gastos conjuntos.

5. Premium

La version Premium desbloquea categorias personalizadas, multiples monedas y reportes avanzados.

6. Ayuda y contacto

Si tienes dudas o problemas, escribenos a soporte@zentavo.it.
