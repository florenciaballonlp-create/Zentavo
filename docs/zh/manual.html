使用手册

快速上手 Zentavo 的指南。

本手册总结了组织财务、记录收支以及使用共享功能的主要步骤。

1. 初始设置

  • 打开 Zentavo 并查看主界面。
  • 在设置中选择主要货币。
  • 如果需要提醒，请开启通知。

2. 记录收入和支出

  • 使用 Income 和 Expenses 按钮添加记录。
  • 填写金额、分类和日期。
  • 保存后查看总余额变化。

3. 储蓄目标

  • 在 Savings 中创建储蓄目标。
  • 设置周期金额并跟踪进度。

4. 共享事件

  • 在 Shared Events 中创建事件并分享代码或二维码。
  • 邀请参与者共同记录支出。

5. Premium

Premium 版本可解锁自定义分类、多货币和高级报表。

6. 帮助与联系

如有问题，请联系 soporte@zentavo.it。

另请参阅 支持、隐私政策 和 使用条款。
