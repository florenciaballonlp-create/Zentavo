User Manual

Quick guide to start using Zentavo.

This manual summarizes the main steps to organize your finances, track movements and use shared features.

1. Initial setup

  • Open Zentavo and review the main dashboard.
  • Set your primary currency in settings.
  • Enable notifications if you want reminders.

2. Track income and expenses

  • Use Income and Expenses to add entries.
  • Complete amount, category and date.
  • Save and check impact in total balance.

3. Savings goals

  • Create goals from the Savings section.
  • Assign periodic amounts and monitor progress.

4. Shared events

  • In Shared Events create an event and share code or QR.
  • Invite participants to track group expenses.

5. Premium

Premium unlocks custom categories, multiple currencies and advanced reports.

6. Help and contact

If you need help, write to soporte@zentavo.it.
