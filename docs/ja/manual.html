ユーザーマニュアル

Zentavo を始めるためのクイックガイドです。

このマニュアルでは、家計管理、収支の記録、共有機能の使い方をまとめています。

1. 初期設定

  • Zentavo を開いてホーム画面を確認します。
  • 設定でメイン通貨を選択します。
  • 必要であれば通知を有効にします。

2. 収入と支出の記録

  • Income と Expenses を使って記録を追加します。
  • 金額、カテゴリ、日付を入力します。
  • 保存後、合計残高への影響を確認します。

3. 貯蓄の利用

  • Savings セクションで目標を作成します。
  • 定期金額を設定し、進捗を確認します。

4. 共有イベント

  • Shared Events でイベントを作成し、コードまたは QR を共有します。
  • 参加者を招待して共同支出を管理します。

5. Premium

Premium では、カスタムカテゴリ、複数通貨、高度なレポートが利用できます。

6. ヘルプと連絡先

ご質問は soporte@zentavo.it までご連絡ください。

サポート、プライバシー、利用規約 もご確認ください。
