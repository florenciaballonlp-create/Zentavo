# Activar GitHub Pages - Instrucciones

## 🎯 Objetivo

Publicar tu Política de Privacidad en línea para cumplir con los requisitos de Google Play Store.

**URL final:** https://florenciaballonlp-create.github.io/Zentavo/

---

## 📋 Pasos para Activar GitHub Pages

### 1. Abre GitHub en tu navegador

Ve a: **https://github.com/florenciaballonlp-create/Zentavo**

### 2. Accede a Configuración (Settings)

1. Busca la pestaña **"Settings"** en la parte superior
2. Click en **Settings**

```
[Code]  [Issues]  [Pull requests]  [Actions]  [Settings] ← Click aquí
```

### 3. Busca la sección Pages

En el menú lateral izquierdo:
1. Desplázate hacia abajo
2. Busca **"Pages"** en la sección "Code and automation"
3. Click en **Pages**

```
📂 Sidebar:
   General
   Access
   Moderation
   Code and automation:
      Webhooks
      Rules
      Environments
      Codespaces
      Pages ← Click aquí
```

### 4. Configura GitHub Pages

Verás una página con el título **"GitHub Pages"**

**Paso A: Source (Origen)**
1. Bajo "Build and deployment"
2. En el dropdown **"Source"**, selecciona: **Deploy from a branch**

**Paso B: Branch (Rama)**
1. En el dropdown **"Branch"**, selecciona: **main**
2. En el dropdown de carpeta, selecciona: **/docs**
3. Click en el botón **"Save"**

```
Build and deployment
┌─────────────────────────────────────┐
│ Source: Deploy from a branch ▼      │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│ Branch: main ▼  /docs ▼   [Save]   │
└─────────────────────────────────────┘
```

### 5. Espera la Publicación

1. Aparecerá un mensaje que dice: **"GitHub Pages is building your site from the /docs folder in the main branch."**
2. Espera 1-2 minutos
3. Recarga la página (F5)
4. Verás una caja verde con el mensaje:
   ```
   ✅ Your site is live at https://florenciaballonlp-create.github.io/Zentavo/
   ```

### 6. Verifica que Funciona

1. Click en el enlace: **https://florenciaballonlp-create.github.io/Zentavo/**
2. Deberías ver tu Política de Privacidad con el diseño azul de Zentavo
3. Verifica que todo el contenido se muestre correctamente

## 🔧 Solución de Problemas

### Problema: No aparece la opción "Settings"

**Solución:** Asegúrate de estar logueado en GitHub con tu cuenta (florenciaballonlp-create)

### Problema: No puedo seleccionar /docs

**Solución:** 
1. Asegúrate de que los cambios se hayan subido (refresh en GitHub)
2. Verifica que existe la carpeta `docs/` con el archivo `index.html`
3. Si no ves /docs, selecciona primero "main" y espera unos segundos

### Problema: Error 404 al visitar el enlace

**Solución:**
1. Espera 5-10 minutos más (a veces tarda)
2. Verifica que seleccionaste /docs y no /(root)
3. Limpia caché del navegador (Ctrl + F5)

### Problema: La página se ve sin estilos

**Solución:**
1. Verifica que el archivo `docs/index.html` está completo
2. Abre el código fuente de la página web (Ctrl + U)
3. Verifica que tiene la sección `<style>` dentro de `<head>`
